# Codeslaves_VKeducation_Max_Hack
Решение кейса Цифровизация от команды Codeslaves для хакатона от VK education и MAX
# Подробное описание ключевых функций
# Абитуриенты (без авторизации)
Ознакоиться с программами вуза
1. Нажмите кнопку "Ознакомиться с программами ВУЗа"
2. Выберите интересующий факультет из списка
3. Просмотрите список образовательных программ
4. Нажмите на программу для детальной информации
5. Узнайте:
   - Описание программы
   - Бюджетные места
   - Проходной балл прошлого года
   - Стоимость обучения
   - Необходимые предметы ЕГЭ

Подбор программ по баллам ЕГЭ
1. Нажмите кнопку "Куда я пройду по баллам ЕГЭ"
2. Выберите предметы, которые вы сдавали на ЕГЭ
3. Введите ваши баллы по каждому предмету
4. Нажмите "Показать доступные программы"
5. Система покажет:
   - Программы, куда вы проходите по баллам
   - Рейтинг программ по проходным баллам
   - Шансы на бюджетные места
   - Альтернативные варианты

"День открытых дверей"
1. Нажмите кнопку "День открытых дверей"
2. Просмотрите список предстоящих мероприятий
3. Выберите интересующее событие
4. Ознакомьтесь с деталями:
   - Дата и время
   - Факультет и программа
   - Описание мероприятия
   - Количество свободных мест
5. Нажмите "Записаться"
6. Пройдите регистрацию
7. Получите подтверждение записи



# Студенты (требуется авторизация)
"Мое расписание"
1. Авторизуйтесь как студент
2. Нажмите "Мое расписание"
3. Система покажет ваше текущее расписание:
   - Занятия на текущую неделю
   - Время и аудитории
   - Преподаватели
   - Тип недели (четная/нечетная)
4. Получайте автоматические уведомления об изменениях

Цифровая кафедра
1. Нажмите "Записаться на цифровую кафедру"
2. Система проверит ваш средний балл (GPA)
3. Покажет доступные направления:
   - Название направления
   - Описание
   - Минимальный требуемый GPA
   - Количество свободных мест
   - Дата окончания приема заявок
4. Выберите направление и подайте заявку
5. Следите за статусом в "Мои заявки на цифровую кафедру"
6. Получите уведомление о решении по заявке

"Создать проект"
1. Нажмите "Создать проект"
2. Введите название проекта (минимум 3 символа)
3. Добавьте подробное описание проекта (минимум 10 символов)
4. Укажите нужные роли в команде (например: бэкенд-разработчик, дизайнер)
5. Подтвердите создание проекта
6. Получите подтверждение:
   - Проект опубликован
   - Другие студенты могут подавать заявки
   - Вы становитесь создателем проекта

"Присоединиться к проекту"
1. Нажмите "Присоединиться к проекту"
2. Просмотрите список доступных проектов:
   - Название проекта
   - Создатель проекта
   - Размер команды
   - Дата создания
3. Выберите интересующий проект
4. Ознакомьтесь с деталями:
   - Полное описание
   - Нужные роли
   - Текущий состав команды
5. Нажмите "Присоединиться"
6. Ожидайте рассмотрения заявки создателем
7. Получите уведомление о решении

"Мои проекты"
1. Нажмите "Мои проекты"
2. Просмотрите два раздела:
   - "Созданные мной проекты" (если вы создатель)
   - "Проекты, в которых я участвую" (как участник)
3. Для созданных проектов:
   - Управляйте заявками других студентов
   - Принимайте/отклоняйте заявки
   - Просматривайте состав команды
4. Для проектов участия:
   - Смотрите свою роль

"Найти книгу"
1. Нажмите "Найти книгу"
2. Введите название книги или автора
3. Просмотрите результаты поиска:
   - Название и автор
   - Доступность (количество свободных экземпляров)
   - Тип книги (бумажная/цифровая)
4. Для бумажных книг:
   - Нажмите "Забронировать"
   - Получите подтверждение бронирования
   - Заберите книгу в библиотеке
5. Для цифровых книг:
   - Нажмите "Читать онлайн"
   - Получите ссылку для чтения
   - Читайте в любое время

"Оформить справку в деканате"
1. Нажмите "Оформить справку в деканате"
2. Выберите тип нужной справки
3. Заполните необходимые данные
4. Отправьте заявку
5. Отслеживайте статус обработки
6. Получите уведомление о готовности
7. Скачайте справку по предоставленной ссылке

# Преподаватели (требуется авторизация)
"Мои занятия"
1. Авторизуйтесь как преподаватель
2. Нажмите "Мои занятия"
3. Просмотрите ваше расписание:
   - Группы и предметы
   - Время и аудитории
   - Тип недели (четная/нечетная)
4. Получайте уведомления об изменениях в расписании

"Оформить командировку"
1. Нажмите "Оформить командировку"
2. Заполните данные командировки:
   - Цель командировки
   - Дата начала
   - Дата окончания
   - Место назначения
3. Отправьте заявку на рассмотрение декану
4. Отслеживайте статус заявки
5. Получите уведомление о решении:
   -Одобрена
   -Отклонена

"Оформить отпуск"
1. Нажмите "Оформить отпуск"
2. Укажите данные отпуска:
   - Дата начала отпуска
   - Дата окончания
   - Количество дней
3. Отправьте заявку на рассмотрение ректору
4. Следите за статусом заявки
5. Получите уведомление о решении


"Конкурс на замещение вакантных должностей"
1. Нажмите "Конкурс на замещение вакантных должностей"
2. Просмотрите доступные вакансии
3. Ознакомьтесь с требованиями
4. Ознакомтесь с текущими контрактами

# Ректор(требуется авторизация)
"Дашборд университета"
1. Авторизуйтесь как ректор
2. Нажмите "Дашборд университета"
3. Просмотрите комплексную статистику:
   - Общее количество студентов и преподавателей
   - Средняя успеваемость (GPA)
   - Количество образовательных программ
   - Статистика по проектной деятельности
   - Данные по цифровой кафедре
   - Регистрации на дни открытых дверей
   - Статистика библиотеки
   - Административные заявки (командировки, отпуска)
4. Используйте "Обновить статистику" для актуальных данных

"Последние новости"
1. Нажмите "Последние новости"
2. Просмотрите новости об НГТУ(ВУЗ взят для примера):
   - Заголовок и источник
   - Дата публикации
   - Тональность (позитивная/нейтральная/негативная)
   - Ссылка на полную версию
3. Мониторьте упоминания университета в СМИ


# Общие функции для всех авторизованных пользователей
"Уведомления"
1. Нажмите "Уведомления"
2. Просмотрите все непрочитанные уведомления
3. Типы уведомлений:
   - Изменения в расписании
   - Статус заявок (проекты, цифровая кафедра)
   - Решения по командировкам и отпускам
   - Готовность справок
   - Библиотечные уведомления
4. Уведомления автоматически помечаются как прочитанные

"Выйти"
1. Нажмите "Выйти"
2. Подтвердите выход из системы
3. Сессия будет завершена
4. Для доступа к персональным функциям потребуется повторная авторизация
